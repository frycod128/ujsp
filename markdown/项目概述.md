## 项目概述

这是一个使用Spring Boot 3.5.9 + JSP构建的Web应用，支持JSP页面渲染和自动打开浏览器功能。

## 项目结构

```
D:.
├───src/main
│   ├───java/fry/ujsp
│   │   ├───UjspApplication.java          # Spring Boot主启动类
│   │   └───controller/JspController.java # 控制器
│   ├───resources
│   │   ├───application.properties        # 配置文件
│   │   ├───static                        # 静态资源
│   │   └───templates                     # 模板文件
│   └───webapp/WEB-INF/jsp                # JSP页面存放目录
......
```

## 主要功能

### 1. 控制器功能 (JspController.java)
提供了两个预设请求映射：
- `/` - 首页
- `/hello` - 示例页面，展示数据传递

### 2. 配置文件 (application.properties)
```properties
spring.mvc.view.prefix=/WEB-INF/jsp/
  # JSP页面前缀
spring.mvc.view.suffix=.jsp
  # JSP页面后缀
server.servlet.jsp.init-parameters.development=true
  # JSP开发模式
server.port=8080
  # 服务器端口
```

### 3. 启动类特性 (UjspApplication.java)
- 继承`SpringBootServletInitializer`支持WAR包部署
- 启动后自动打开浏览器访问`http://localhost:8080`
- 支持多操作系统（Windows/Mac/Linux）的自动打开功能

## 技术栈

- **Spring Boot 3.5.9** - 核心框架
- **Java 17** - JDK版本
- **JSP** - 视图层技术
- **JSTL 3.0** - JSP标签库
- **Maven** - 构建工具
- **Embedded Tomcat** - 内嵌服务器

## 依赖说明

主要依赖包括：
- `spring-boot-starter-web` - Web基础依赖
- `tomcat-embed-jasper` - JSP解析支持
- `jakarta.servlet.jsp.jstl` - JSTL支持
- `jakarta.servlet-api` - Servlet API

## 使用说明

1. **运行项目**：直接运行`UjspApplication.java`
2. **访问页面**：程序会自动打开浏览器，或手动访问`http://localhost:8080`

## 注意事项

1. 确保在`/src/main/webapp/WEB-INF/jsp/`目录下创建对应的JSP文件
2. 项目打包类型为WAR，但可以以JAR方式运行（内嵌Tomcat）
3. JSP页面需要使用JSTL标签库时需要引入taglib指令

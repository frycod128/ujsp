在Spring Boot中同时接收多种类型的POST参数，有以下几种常用方式：

## 1. 混合使用@RequestParam和@RequestBody

```java
@RestController
@RequestMapping("/api")
public class MixedParamsController {
    
    @PostMapping("/mixed1")
    public Map<String, Object> handleMixedParams(
            @RequestParam("username") String username,              // 表单参数
            @RequestParam(value = "age", required = false) Integer age,
            @RequestBody(required = false) Map<String, Object> jsonBody,  // JSON参数
            @RequestHeader("User-Agent") String userAgent,          // 请求头
            @CookieValue(value = "JSESSIONID", required = false) String sessionId // Cookie
    ) {
        Map<String, Object> result = new HashMap<>();
        result.put("username", username);
        result.put("age", age);
        result.put("jsonBody", jsonBody);
        result.put("userAgent", userAgent);
        result.put("sessionId", sessionId);
        result.put("timestamp", LocalDateTime.now());
        return result;
    }
}
```

## 2. 使用HttpServletRequest直接获取所有参数

```java
@Controller
public class JspController {
    
    @PostMapping("/all-params")
    public String handleAllParams(HttpServletRequest request, Model model) {
        // 获取所有参数
        Map<String, String[]> parameterMap = request.getParameterMap();
        Map<String, String> singleValueMap = new HashMap<>();
        
        // 转换参数（取每个参数的第一个值）
        parameterMap.forEach((key, values) -> {
            if (values.length > 0) {
                singleValueMap.put(key, values[0]);
            }
        });
        
        // 获取请求体（适用于JSON、XML等）
        String requestBody = null;
        try {
            requestBody = request.getReader().lines()
                    .collect(Collectors.joining(System.lineSeparator()));
        } catch (IOException e) {
            e.printStackTrace();
        }
        
        // 获取请求头
        Enumeration<String> headerNames = request.getHeaderNames();
        Map<String, String> headers = new HashMap<>();
        while (headerNames.hasMoreElements()) {
            String headerName = headerNames.nextElement();
            headers.put(headerName, request.getHeader(headerName));
        }
        
        // 添加到Model
        model.addAttribute("parameters", singleValueMap);
        model.addAttribute("requestBody", requestBody);
        model.addAttribute("headers", headers);
        model.addAttribute("method", request.getMethod());
        model.addAttribute("contentType", request.getContentType());
        
        return "params-display";
    }
}
```

            @RequestPart("formData") FormData formData,      // multipart/form-data 表单
            @RequestPart("jsonData") JsonData jsonData,      // JSON数据部分
            @RequestPart(value = "file", required = false) MultipartFile file, // 文件
            @RequestParam("type") String type,                // URL参数
            @RequestHeader("X-Request-ID") String requestId   // 请求头
    ) {
        Map<String, Object> response = new HashMap<>();
        response.put("formData", formData);
        response.put("jsonData", jsonData);
        response.put("type", type);
        response.put("requestId", requestId);
        
        if (file != null && !file.isEmpty()) {
            response.put("fileName", file.getOriginalFilename());
            response.put("fileSize", file.getSize());
        }
        
        return response;
    }
}

## 5. 使用@RequestPart接收混合类型数据

```java
@PostMapping(value = "/mixed-data", consumes = MediaType.MULTIPART_FORM_DATA_VALUE)
public ResponseEntity<Map<String, Object>> handleMixedData(
        @RequestPart("user") User user,                    // JSON数据
        @RequestPart("profile") UserProfile profile,       // JSON数据
        @RequestPart(value = "avatar", required = false) MultipartFile avatar,  // 文件
        @RequestParam("action") String action,             // URL参数
        @RequestHeader("Authorization") String token       // 请求头
) {
    Map<String, Object> result = new HashMap<>();
    result.put("user", user);
    result.put("profile", profile);
    result.put("action", action);
    result.put("token", token.substring(0, Math.min(10, token.length())) + "...");
    
    if (avatar != null && !avatar.isEmpty()) {
        result.put("avatarName", avatar.getOriginalFilename());
        result.put("avatarSize", avatar.getSize());
    }
    
    return ResponseEntity.ok(result);
}
```

## 8. 完整Controller示例

```java
@RestController
@RequestMapping("/api/advanced")
public class AdvancedParamsController {
    
    private static final Logger logger = LoggerFactory.getLogger(AdvancedParamsController.class);
    
    /**
     * 同时接收多种类型的参数
     */
    @PostMapping(value = "/process", consumes = {MediaType.APPLICATION_JSON_VALUE, MediaType.MULTIPART_FORM_DATA_VALUE})
    public ResponseEntity<ApiResponse> processAdvancedParams(
            // 1. URL参数
            @RequestParam String action,
            @RequestParam(defaultValue = "1") Integer version,
            
            // 2. 请求头
            @RequestHeader("User-Agent") String userAgent,
            @RequestHeader(value = "X-Client-ID", required = false) String clientId,
            
            // 3. Cookie
            @CookieValue(value = "sessionId", required = false) String sessionId,
            
            // 4. JSON请求体
            @RequestBody(required = false) Map<String, Object> jsonData,
            
            // 5. 文件上传
            @RequestPart(value = "file", required = false) MultipartFile file,
            
            // 6. HttpServletRequest（获取所有信息）
            HttpServletRequest request
    ) {
        ApiResponse response = new ApiResponse();
        response.setSuccess(true);
        response.setTimestamp(LocalDateTime.now());
        
        // 收集所有参数
        Map<String, Object> receivedData = new LinkedHashMap<>();
        
        // URL参数
        receivedData.put("action", action);
        receivedData.put("version", version);
        
        // 请求头
        receivedData.put("userAgent", userAgent);
        receivedData.put("clientId", clientId);
        receivedData.put("sessionId", sessionId);
        
        // JSON数据
        if (jsonData != null && !jsonData.isEmpty()) {
            receivedData.put("jsonData", jsonData);
        }
        
        // 文件信息
        if (file != null && !file.isEmpty()) {
            Map<String, Object> fileInfo = new HashMap<>();
            fileInfo.put("name", file.getOriginalFilename());
            fileInfo.put("size", file.getSize());
            fileInfo.put("contentType", file.getContentType());
            receivedData.put("file", fileInfo);
        }
        
        // 获取所有请求参数
        Map<String, String[]> parameterMap = request.getParameterMap();
        if (!parameterMap.isEmpty()) {
            receivedData.put("allParams", parameterMap);
        }
        
        // 获取所有请求头
        Map<String, String> headers = new HashMap<>();
        Collections.list(request.getHeaderNames()).forEach(name -> 
            headers.put(name, request.getHeader(name))
        );
        receivedData.put("allHeaders", headers);
        
        response.setData(receivedData);
        response.setMessage("成功接收" + receivedData.size() + "种参数");
        
        logger.info("收到请求: {}", response);
        
        return ResponseEntity.ok(response);
    }
    
    // 内部响应类
    static class ApiResponse {
        private boolean success;
        private LocalDateTime timestamp;
        private String message;
        private Object data;
        
        // getters and setters
        public boolean isSuccess() { return success; }
        public void setSuccess(boolean success) { this.success = success; }
        public LocalDateTime getTimestamp() { return timestamp; }
        public void setTimestamp(LocalDateTime timestamp) { this.timestamp = timestamp; }
        public String getMessage() { return message; }
        public void setMessage(String message) { this.message = message; }
        public Object getData() { return data; }
        public void setData(Object data) { this.data = data; }
    }
}
```

## 总结

同时接收多种参数的主要方式：

1. **组合注解**：同时使用 `@RequestParam`、`@RequestBody`、`@RequestHeader`、`@CookieValue`
2. **HttpServletRequest**：直接获取所有请求信息
3. **@RequestPart**：处理 multipart 请求中的不同类型数据
4. **自定义参数对象**：封装不同类型的参数
5. **Map/JsonNode**：动态接收不确定结构的参数

选择哪种方式取决于您的具体需求：
- 参数确定且类型不同：使用组合注解
- 需要灵活处理：使用 HttpServletRequest
- 包含文件上传：使用 @RequestPart
- 参数结构复杂：使用自定义对象
